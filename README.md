# Prerequisite

- Familiarity with `bash` and Python
- CERN lxplus account
- Search and subscribe to the following email groups [here](https://e-groups.cern.ch/e-groups/EgroupsSearchForm.do)
    - for read access to FCC event samples: `fcc-eos-access`
    - for access to FCC computing resources (condor): `fcc-experiments-comp`
